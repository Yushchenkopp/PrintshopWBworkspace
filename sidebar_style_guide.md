# Руководство по стилю сайдбара (Glassmorphism)

Этот стиль представляет собой премиальную реализацию эффекта «матового стекла» (Glassmorphism), вдохновленного интерфейсами Apple. Основной акцент сделан на физичности материала, преломлении света на гранях и глубоком размытии фона.

## 1. Основные параметры (Core Glass)

Стиль строится на сочетании высокого размытия (`backdrop-filter`) и частичной прозрачности белого фона.

*   **Размытие (Blur):** `backdrop-blur: 40px` (в Tailwind: `backdrop-blur-2xl`). Это обеспечивает полную потерю четкости объектов под сайдбаром, превращая их в мягкие цветовые пятна.
*   **Цвет фона:** `rgba(255, 255, 255, 0.6)` (Белый с 60% непрозрачности). Такая плотность позволяет цветам снизу «просачиваться» сквозь панель, но сохраняет достаточную контрастность для чтения текста.
*   **Геометрия:** `border-radius: 24px` (`rounded-3xl`). Крупные скругления создают современный и «дружелюбный» вид.

## 2. Эффект преломления на гранях (Edge Refraction / Rim Light)

Это самая тонкая деталь, которая придает стеклу объем. Вместо однородной рамки используются разные значения непрозрачности для каждой стороны.

```css
border-width: 1px;
border-style: solid;
/* Порядок: Top, Right, Bottom, Left */
border-color: 
  rgba(255, 255, 255, 0.7) /* Верх: блик от источника света */
  rgba(255, 255, 255, 0.3) /* Право: менее выраженная грань */
  rgba(255, 255, 255, 0.3) /* Низ: затененная грань */
  rgba(255, 255, 255, 0.7); /* Лево: блик от источника света */
```

**Зачем это нужно:** Создается имитация падения света слева-сверху. Белые грани сверху и слева выглядят как яркие блики на ребре стекла, а более прозрачные грани снизу и справа создают ощущение объема.

## 3. Тень и глубина (Elevation)

Для отделения стекла от остального контента используется мягкая, очень широкая тень:

*   **Тень:** `shadow-2xl` (в Tailwind). Это примерно `box-shadow: 0 25px 50px -12px rgba(0, 0, 0, 0.25);`. Она не имеет четких границ и создает эффект «парения» над холстом.

## 4. Сводный CSS (Checklist)

Если вы хотите применить этот стиль в другом проекте, используйте следующие свойства:

```css
.glass-sidebar {
  /* Размеры и позиционирование */
  width: 400px;
  max-height: calc(100vh - 48px);
  position: fixed;
  left: 24px;
  top: 24px;
  
  /* Эффект матового стекла */
  background-color: rgba(255, 255, 255, 0.6);
  backdrop-filter: blur(40px);
  -webkit-backdrop-filter: blur(40px); /* Для поддержки Safari */
  
  /* Границы (Rim Light) */
  border-width: 1px;
  border-style: solid;
  border-color: rgba(255, 255, 255, 0.7) rgba(255, 255, 255, 0.3) rgba(255, 255, 255, 0.3) rgba(255, 255, 255, 0.7);
  
  /* Скругление и тень */
  border-radius: 24px;
  box-shadow: 0 25px 50px -12px rgba(0, 0, 0, 0.25);
  
  /* Улучшение производительности */
  transform: translateZ(0);
  will-change: transform, backdrop-filter;
}
```

## 5. Взаимодействие с цветом (Refraction)

Эффект «преломления цветов» под слоем сайдбара достигается автоматически за счет сочетания:
1.  **Высокого размытия** (`backdrop-blur`), которое смешивает пиксели фона.
2.  **Низкой плотности фона** (0.6), которая позволяет яркости фоновых объектов влиять на итоговый цвет панели.

Если под сайдбаром находятся яркие фотографии или градиенты (как в разделе «Коллаж»), они будут «светиться» сквозь панель, создавая динамичный адаптивный интерфейс.
